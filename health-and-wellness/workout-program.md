# Workout Program

## Program Philosophy
- Goal hierarchy: consistency first, fat loss second, performance improvement third
- Form before load
- Conservative progressive overload
- Protect lower back and left shoulder
- Build a repeatable habit, not hero workouts

## Block 1 Overview
- Block length: 4 weeks
- Split: 4 gym days + 1–2 run/recovery days
- Gym split: Upper A / Lower A / Upper B / Lower B
- Built-in standing components: core stability, rotator cuff/scapular work, movement prep
- Reassessment: end of week 4 using logged data

## Weekly Template
- Monday — Upper A
- Tuesday — Lower A + core
- Wednesday — optional recovery / mobility
- Thursday — Upper B + core
- Friday — Lower B + core
- Saturday — full rest
- Sunday — run
- Optional second easy run: Wednesday if recovery is good

### Day 1 — Upper A
- Shoulder/scap warm-up
- Main upper push
- Main upper pull
- Secondary push
- Secondary pull
- Rotator cuff / scap work
- Core stability finisher

### Day 2 — Lower A
- Dynamic warm-up
- Main squat pattern
- Main hinge pattern (back-safe)
- Single-leg accessory
- Hamstring accessory
- Calf / lower-leg work optional
- Core stability finisher

### Day 3 — Run / Recovery
- Easy outdoor run or incline walk
- Mobility work
- Optional soft tissue / shoulder care

### Day 4 — Upper B
- Shoulder/scap warm-up
- Alternate upper push
- Alternate upper pull
- Secondary chest/shoulder/triceps
- Secondary back/biceps
- Rotator cuff / scap work
- Core stability + controlled rotation

### Day 5 — Lower B
- Dynamic warm-up
- Main leg press / squat variation
- Posterior chain movement
- Quad accessory
- Glute accessory
- Optional conditioning finisher
- Core stability finisher

### Day 6 — Run / Recovery
- Easy/moderate run based on recovery
- Mobility / tissue work

### Day 7 — Rest
- Walk, light mobility, nothing stupid

## Progression Rules
- Week 1: establish technique and honest baseline loads
- Week 2: small load or rep progression if prior session was within target RPE and form was stable
- Week 3: repeat progression conservatively
- Week 4: hold or slightly push where earned, then review data

### Load progression default
- Upper body compounds: add the smallest sensible increment only if all prescribed work was completed with clean form and target RPE
- Lower body compounds: same rule, conservative jumps only
- Accessories: progress reps before load when possible
- Running: progress mostly through consistency and small distance/time improvements, not ego pace chasing

## Initial Exercise Selection
### Upper A
1. Cable chest press or machine chest press
2. Seated cable row
3. Incline dumbbell press or machine incline press
4. Lat pulldown
5. Cable lateral raise
6. Cable external rotation / face pulls
7. Pallof press

### Lower A
1. Leg press or goblet squat
2. Romanian deadlift with dumbbells or machine hinge alternative
3. Split squat or step-up
4. Seated or lying leg curl
5. Standing calf raise
6. Dead bug / bird dog / McGill-style core work

### Upper B
1. Machine shoulder press or landmine press (shoulder tolerance dependent)
2. Chest-supported row
3. Cable fly or pec deck
4. Neutral-grip pulldown or assisted pull-up variation
5. Cable triceps pressdown
6. Cable curl
7. Woodchop + side plank pairing

### Lower B
1. Hack squat / pendulum squat / leg press variation
2. Hip thrust or glute bridge machine
3. Leg extension
4. Hamstring curl variation
5. Walking lunge if tolerated
6. Farmer carry or suitcase carry (only if back-safe) / core alternative

## Target Effort
- Most sets: RPE 6–8
- Avoid grinders early
- Stop any movement that causes sharp pain, nerve symptoms, or shoulder pinching
